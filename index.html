Deployment & Play Store Publishing Guide

How do I host this on a free domain?

You can host this website completely for free using GitHub Pages, Netlify, or Vercel.

  1. Create a free account on GitHub.
  2. Create a new repository named vela.
  3. Upload the files (index.html, style.css, app.js, manifest.json, sw.js, and the icons).
  4. Go to repository Settings -> Pages.
  5. Select the main branch and click Save.
  6. Within 1-2 minutes, your dashboard will be live at https://<your-username>.github.io/vela/ for free!
How do I build this into a mobile app for Google Play Store?

Since Vela is configured as a fully compliant Progressive Web App (PWA), you can package it into an Android app package (.apk / .aab) to upload to the Play Store for free.

  1. Deploy your app to a free domain (e.g. your GitHub Pages domain).
  2. Visit PWABuilder.com.
  3. Paste your GitHub Pages URL and click Start.
  4. PWABuilder will automatically scan your manifest file and package details.
  5. Click Generate App Package -> Android (Google Play).
  6. Download your signed .aab package and upload it directly to your Google Play Console!
